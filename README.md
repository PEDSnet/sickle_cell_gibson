# Impact of Iron Overload on Hematopoietic Stem Cell Transplant in Children with Non-Malignant Hematological Disorders
---

**PIs**: Nora Gibson

**Data Scientist/Biostatistician**: Nhat Nguyen

This repository contains all the codes code and codesets for cohort development, data quality check, chart review sampling and validation, and statistical analysis for the ["Impact of Iron Overload on Hematopoietic Stem Cell Transplant in Children with Non-Malignant Hematological Disorders" manuscript](https://doi.org/10.1182/bloodadvances.2025017942).
